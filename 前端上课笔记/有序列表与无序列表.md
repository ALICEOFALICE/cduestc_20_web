# 有序列表与无序列表

## 列表分为两种：

	* 有序列表
	* 无序列表

---

## 有序列表

- 有序列表语法为

```html
<ol>
    <li></li>
    <li></li>
    <li></li>
</ol>
```

`<ol>`是不放东西的，如果要放东西就多嵌套一层

如

```html
<ol>
    <li>成都
        <Ol>
            <li>金牛区</li>
            <li>郫都区</li>
        </Ol>
    </li>
</ol>
```

### 无序列表

- 无序列表的语法为

  ```html
  <ul>
      <li></li>
      <li></li>
      <li></li>
  </ul>
  ```

  `<ul>`同样不能放内容，请自行参考上文

## 折叠标签

- 折叠标签分为两个部分，缺一不可。

  - `summary`

    这个标签是展开标签最外层显示的文字

  - `details`

    这个标签用于将需要折叠的内容框起来，除了`summary`标签外的所有内容均会被折叠

- 例子

  ```html
  <details>
      <summary>这是能显示出来的</summary>
      <li>这是显示不出来的（被折叠了）</li>
  </details>
  ```
