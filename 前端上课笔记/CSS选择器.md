# CSS选择器

## 属性选择器

> 属性选择器可以根据元素的属性和属性值来选择属性。CSS3中增加了3中属性选择器。

```css
p[id^=one]{
    color="red"
}
```

这个P标签是指的P标签内带有id指示器且id指示器值为one后执行

需要同时满足以下要求后方可启用4

- 需要是`P`标签
- 需要有ID指示器
- 需要ID指示器的值为`one`





### 普通兄弟元素

```html
<head>
<style>
    p~h2{
        color:pink;
    }
</style>
</head>
<body>
    <p>这个应该不会显示颜色</p>
    <h2>这个大概率会变成樱花粉</h2>
</body>
```
